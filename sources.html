Methodological Tools

  • fNIRS Hardware: fNIR Imager 1000 (fNIR Devices LLC) with 16-channel forehead sensor array.
  • Data Acquisition: COBI Studio (v.1.5.0.55) for real-time hemodynamic quantification.
  • Processing Pipeline: MNE-Python & SciPy for Butterworth band-pass filtering (0.01–0.2 Hz) and spike interpolation.
  • Statistical Modeling: Linear Mixed-Effects (LME) models with participant-level random intercepts (statsmodels v.0.14).

Music Generation Sources

  • AI Music: Generated using Suno v4.0 and Udio AI platforms, customized via prompt engineering to match genre-specific emotional parameters.
  • Human Music: Curated from independent artists (Creative Commons) and studio recordings, normalized for loudness and frequency response.
